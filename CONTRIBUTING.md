# 贡献指南

感谢你对 OrangePi Debug Tool 项目的关注！

## 🚀 快速开始

```bash
# 克隆仓库
git clone https://github.com/xfengyin/orangepi-debug-tool.git
cd orangepi-debug-tool

# 安装开发依赖
pip install -e ".[dev]"

# 运行测试
pytest tests/ -v

# 运行应用
orangepi-debug
```

## 📝 代码规范

- 使用 Black 格式化代码
- 遵循 PEP 8 规范
- 添加类型注解
- 编写单元测试
