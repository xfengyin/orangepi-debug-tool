## 🚀 快速入门

### 1. 启动应用

```bash
# 开发模式
npm run tauri dev

# 或双击安装的应用程序
```

### 2. 连接串口

1. 选择串口端口 (如 `/dev/ttyUSB0`)
2. 设置波特率 (默认 115200)
3. 点击"连接"按钮

### 3. 发送数据

1. 在输入框输入数据
2. 选择发送模式 (文本/Hex)
3. 点击"发送"或按 Enter

## 📡 串口调试

### 串口配置

| 参数 | 选项 | 说明 |
|------|------|------|
| 波特率 | 1200-921600 | 通信速率 |
| 数据位 | 5/6/7/8 | 数据位数 |
| 校验位 | None/Even/Odd/Mark/Space | 校验方式 |
| 停止位 | 1/1.5/2 | 停止位数 |
| 流控制 | None/Hardware/Software | 流控制方式 |

### 发送选项

- **Hex 模式** - 发送十六进制数据
- **添加 LF** - 自动添加换行符
- **添加 CR** - 自动添加回车符
- **显示时间戳** - 显示消息时间

### 统计信息

- 发送字节数
- 接收字节数
- 发送帧数
- 接收帧数
- 错误数

### 快捷命令

1. 点击"添加命令"
2. 输入命令名称和数据
3. 设置定时发送间隔 (可选)
4. 点击保存

## 🔍 协议解析

### MODBUS RTU

**支持功能：**
- 读取保持寄存器 (功能码 03)
- 写入单个寄存器 (功能码 06)
- 写入多个寄存器 (功能码 16)

**帧格式：**
```
[从站地址][功能码][数据][CRC16]
   1 字节    1 字节   N 字节  2 字节
```

### 自定义协议

1. 选择"自定义协议"
2. 配置帧头/帧尾
3. 设置校验方式
4. 定义字段解析规则

## ⚙️ 高级功能

### 多串口管理

**同时连接多个串口：**

1. 点击"添加串口"
2. 选择端口和波特率
3. 设置标签和颜色
4. 最多支持 8 个串口

### 宏命令/脚本

**脚本示例：**

```javascript
// MODBUS 读取脚本
SET slave 1
SET function 03
SET address 0
SET count 10

PRINT Reading ${count} registers...
SEND ${slave} ${function} ${address} ${count}
WAIT 100
PRINT Done
```

**支持命令：**
- `SET` - 设置变量
- `PRINT` - 打印输出
- `SEND` - 发送数据
- `WAIT` - 等待指定时间
- `IF` - 条件判断

### 主题切换

- 浅色主题
- 深色主题
- 跟随系统

### 国际化

- 中文 (简体)
- English

## ❓ 常见问题

### Q: 无法检测到串口？

**A:** 检查以下几点：
1. 确认设备已连接
2. 检查 USB 驱动是否安装
3. 确认用户有串口访问权限
   ```bash
   sudo usermod -a -G dialout $USER
   ```

### Q: 数据乱码？

**A:** 可能原因：
1. 波特率设置不正确
2. 数据位/校验位/停止位不匹配
3. 尝试切换 Hex 模式

### Q: 如何导出聊天记录？

**A:** 
1. 点击"数据日志"标签
2. 选择导出格式 (CSV/JSON/TXT)
3. 点击"导出"按钮

### Q: 脚本不执行？

**A:** 检查：
1. 脚本语法是否正确
2. 变量名是否匹配
3. 查看错误提示信息

### Q: 如何同时连接多个设备？

**A:** 
1. 点击"添加串口"
2. 为每个设备配置独立参数
3. 每个串口有独立颜色标识
