## 升级指南

### 从 v1.x 升级到 v2.0

1. **系统要求更新**
   - Rust 最低版本要求提升至 1.70
   - Node.js 最低版本要求提升至 16

2. **配置文件迁移**
   - 旧版配置文件位于 `~/.orangepi-debug-tool/`
   - 新版配置位于 `~/.config/orangepi-debug-tool/`
   - 请手动迁移您的配置

3. **API 变更**
   - 串口命令命名空间变更：`serial` → `debug.serial`
   - GPIO 命令命名空间变更：`gpio` → `debug.gpio`
   - 请参考新的 API 文档更新您的代码

4. **插件系统**
   - 移除了旧版内置插件系统
   - 改用 Tauri 2.0 官方插件架构
   - 第三方插件需要更新以兼容新架构
