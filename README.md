# OrangePi 上位机调试工具

一款自用的 OrangePi 调试工具，用于简化开发和调试过程。

## 简介

这是一个专为 OrangePi 开发者设计的上位机调试工具，旨在简化硬件调试和测试流程。

## 功能特性

- 实时串口通信
- GPIO 控制
- 数据可视化
- 日志记录
- 设备状态监控

## 安装

```bash
git clone https://github.com/xfengyin/orangepi-debug-tool.git
cd orangepi-debug-tool
# 根据具体实现选择安装方式
```
