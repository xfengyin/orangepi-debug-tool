# 版本发布说明

## v1.0.0 - 初始版本

### 新增功能
- [x] 基础串口通信功能
  - [x] 支持多种波特率选择
  - [x] 实时数据收发
  - [x] 十六进制/文本模式切换
- [x] 图形用户界面
  - [x] 串口连接设置界面
  - [x] 数据发送区域
  - [x] 数据接收区域（带时间戳）
  - [x] GPIO 控制界面
- [x] GPIO 控制功能
  - [x] 通过串口命令控制 GPIO
  - [x] 支持 HIGH/LOW 状态设置
- [x] 配置管理
  - [x] 配置文件支持
  - [x] 默认参数设置
- [x] 文档
  - [x] README.md 项目说明
  - [x] 详细使用指南
  - [x] 配置说明
- [x] 测试
  - [x] 核心逻辑测试
  - [x] 依赖检查

### 技术规格
- Python 3.6+
- pyserial 库
- Tkinter GUI 框架
- 跨平台支持（Windows, macOS, Linux）

### 已知问题
- 在无头环境下无法运行 GUI 测试
- 某些系统可能需要额外权限访问串口

- 增加更多的硬件控制命令
- 实现自动更新功能
